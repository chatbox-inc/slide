Wintersmith

静的サイトジェネレータ。 Hexoのごっつシンプル版みたいな

http://wintersmith.io/

静的サイトジェネレータ触った時によくある、 お作法の学習みたいなのが殆ど無い。とってもシンプル。

ドキュメントも非常に少ない。

ドキュメントも非常に少ない。

Wintersmith Quick Start guide

https://github.com/jnordberg/wintersmith#quick-start

$ npm install wintersmith -gでインストール。

$ wintersmith new <path>でプロジェクト作成

基本的なブログテンプレートのスケルトンが出来上がる。

cdしてwintersmith previewでサイトの確認が可能。

Wintersmith ディレクトリ構成

contentsディレクトリの中のファイルがbuildにそのまま流れていくイメージ

$ tree
.
├── config.json  // 設定ファイル
├── build        // ビルドした際の結果格納ディレクトリ
├── contents     // メインのCSSや画像HTMLなど
├── node_modules
├── package.json
├── plugins      // プラグインスクリプトを配置
├── readme.md
└── templates    // 各種テンプレートファイル(jade)

ビルド時にプラグインを実行したりとか、 yfm,jsonで書かれたオブジェクトをjadeテンプレートに流し込むことが可能
